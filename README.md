# Sistema de Biblioteca

Sistema de gerenciamento de biblioteca desenvolvido em C# para prática de Programação Orientada a Objetos.

## Funcionalidades

- Listar todos os livros cadastrados
- Emprestar livros para usuários
- Controle de status (Disponível/Emprestado)
- Cadastro automático de locatários

## Como executar
```bash
# Clone o repositório
git clone https://github.com/SEU_USUARIO/SistemaBiblioteca.git

# Entre na pasta
cd SistemaBiblioteca

# Execute o projeto
dotnet run
```

## Tecnologias

- C# / .NET 8.0
- Programação Orientada a Objetos

## Estrutura do Projeto

- `Livro.cs` - Classe que representa um livro
- `Pessoa.cs` - Classe que representa um usuário
- `Biblioteca.cs` - Classe principal do sistema
- `Program.cs` - Menu e lógica de interação
